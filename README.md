# Sentinel-X

SentinelX is a modular security ecosystem integrating AI, Machine Learning and Web3 technologies. This project is divided into five specialized repositories.

## Repositories

### 1. Sentinel-X ML
Dedicated to machine learning model training, data processing pipelines and model versioning.
* **Repo:** [ML Repo](https://github.com/ruchit2005/Sentinel-X-ML-Model)

### 2. Sentinel-X Backend
The primary server-side logic, database management and business rules orchestration.
* **Repo:** [Backend Repo](https://github.com/pratham-developer/Sentinel-X)

### 3. Sentinel-X Web3
The Sentinel-X Vault is an immutable registry deployed on the Sepolia Testnet. It acts as the "Root of Trust" for application distribution.
* **Repo:** [Web3 Repo](https://github.com/pratham-developer/SentinelX-Vault)

### 4. Sentinel-X App
Contains the core AI service API built with FastAPI for high-performance inference and data handling.
* **Repo:** [AI FastApiService Repo](https://github.com/ruchit2005/Sentinel-X-Ai-Fastapi-microservice.git)

### 5. Sentinel-X AI FastApiService
The user-facing mobile application for interacting with the SentinelX ecosystem.
* **Repo:** [AI FastApiService Repo](https://github.com/Rudragupta8777/SentinelX_App)

### 5. Sentinel-X Figma
The complete UI/UX design system, prototypes and high-fidelity mockups for the SentinelX platform.
* **Repo:** [Figma File](https://www.figma.com/design/LalpleVmDJTzsf8umPdRub/Sentinel-X?node-id=0-1&t=VjcIXFsQX34YJQf3-1)
